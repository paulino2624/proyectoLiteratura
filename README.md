# proyectoLiteratura

La siguiente aplicación está conectada con la API Gutendex, API relacionada a obras literarias escritas.

Se podrá agregar libros a la base de datos en el que posteriormente se podrán hacer peticiones según el siguiente menú de comandos.

1. Buscar libro por titulo
2. Listar libros registrados
3. Listar autores registrados
4. Listar autores en un determinado año
5. Listar libros por idioma
0. Salir
